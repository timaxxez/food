ИШКАЕВ ТИМУР 9-2ПО-22, ПРИЛОЖЕНИЕ НА FLUTTER

Food Delivery — это мобильное приложение на Flutter, которое позволяет жителям города быстро и удобно заказывать еду из местных кафе и ресторанов. 
Пользователь может просматривать меню, выбирать блюда, оформлять заказ и отслеживать его статус в реальном времени. 
Приложение имеет простой и понятный интерфейс, ориентированный на скорость и комфорт использования.
